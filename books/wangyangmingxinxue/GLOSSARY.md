# 共享术语词典

本词典整理自 `candidates/glossary.md`，供所有 skill 统一使用。

| 术语 | 蒸馏后的工作定义 | 关键边界 |
|---|---|---|
| 四句教 | 心、意、良知、格物组成的念头处理链。 | 不等于道德虚无或玄学口号。 |
| 良知 | 面对具体情境时知善知恶的价值判断能力。 | 不等于情绪、本能或第一反应。 |
| 致良知 | 让判断显明，并落实为具体行动。 | 不能跳过事实、专业和程序校验。 |
| 格物 | 在事情与念头上正其不正。 | 不等于拒绝外部研究或科学调查。 |
| 知行合一 | 真知须由行动完成，行动反过来检验理解。 | 不鼓励盲动或跳过必要准备。 |
| 事上磨炼 | 在真实职责、冲突和压力中训练、校正。 | 不把危险、创伤或受害场景浪漫化。 |
| 私欲 | 使判断偏私、逃避或执著得失的名利、恐惧、习气与虚荣。 | 不把正常需求、情感和利益一概道德化。 |
| 立志 | 用长期方向校正日常投入与念头。 | 不等于执拗的单一人生目标。 |
| 责善 | 先反己、再以帮助改正为目的反馈。 | 不适用于要求受害者自责或替代正式处置。 |
